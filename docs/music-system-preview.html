🔀 Crossfade בין זירות
מעבר חלק בין רצועות כשהשחקן עובר בין מסכים.
Equal-Power Crossfade
  • משך: 1.5 שניות (configurable)
  • עקומה: linearRamp — fade out ישן + fade in חדש
  • Buffer: הרצועה החדשה נטענת מראש (preload)
  • דה-דופליקציה: אם אותה זירה — skip
  • ניקוי: source ישן מתנתק אחרי fade + 100ms
Stinger Ducking
  • Duck level: musicGain יורד ל-30%
  • Fade time: 0.4 שניות (חלק, לא חד)
  • שחזור: אוטומטי כשהג'ינגל נגמר
  • דוגמה: Victory fanfare → רקע יורד → fanfare נגמר → רקע חוזר
  • כפילות: אם 2 stingers רצים, duck נשמר עד שהאחרון נגמר
Crossfade Implementation
// Equal-power crossfade between zones const fadeDuration = 1.5; // seconds const now = audioContext.currentTime; // Fade out current track currentGain.gain.setValueAtTime(currentGain.gain.value, now); currentGain.gain.linearRampToValueAtTime(0, now + fadeDuration); // Fade in new track newGain.gain.setValueAtTime(0, now); newGain.gain.linearRampToValueAtTime(1, now + fadeDuration); // Stop old source after fade completes currentSource.stop(now + fadeDuration + 0.1); // New source starts immediately (already loaded via preload) newSource.start(0);

Component example
import { useMusic } from '@/components/audio/music-provider' function BattleScreen() { const { playZone, playStinger, stopAll } = useMusic() // Switch to battle music const startBattle = () => { playZone('battle-arena') } // Play victory jingle (ducks background automatically) const onVictory = () => { playStinger('victory-fanfare') } // Stop all music const onExit = () => { stopAll() } return ( <div> <button onClick={startBattle}>התחל קרב</button> <button onClick={onVictory}>ניצחון!</button> <button onClick={onExit}>יציאה</button> </div> ) }

import { useMusicStore } from '@/stores/music-store' function MusicSettings() { const volume = useMusicStore((s) => s.musicVolume) const muted = useMusicStore((s) => s.musicMuted) const enabled = useMusicStore((s) => s.musicEnabled) const setVolume = useMusicStore((s) => s.setMusicVolume) const toggleMute = useMusicStore((s) => s.toggleMusicMute) const toggleEnabled = useMusicStore((s) => s.toggleMusicEnabled) return ( <div> <label> <input type="checkbox" checked={enabled} onChange={toggleEnabled} /> מוזיקה מופעלת </label> <label> <input type="checkbox" checked={muted} onChange={toggleMute} /> השתקת מוזיקה </label> <input type="range" min={0} max={1} step={0.05} value={volume} onChange={(e) => setVolume(Number(e.target.value))} /> </div> ) }
