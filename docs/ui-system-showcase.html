🛠 עקרונות עיצוב

כללים שכל ה-utilities למעלה מקיימים.

✅ CSS Logical Properties — ms- ולא ml-, ps- ולא pl-, inset-inline-start ולא left
✅ CSS Custom Properties — כל הערכים דרך --game-* variables לקסטומיזציה
✅ transform/opacity Only — כל האנימציות רק transform ו-opacity (GPU accelerated)
✅ RTL-First — כל ה-utilities תומכים ב-[dir='rtl'] באופן מובנה
✅ prefers-reduced-motion — כל האנימציות מושבתות כש-reduce מופעל
✅ Light Theme — כל ה-utilities כוללים .light overrides
✅ Max 200ms — כל ה-transitions עד 150ms, animations עד 300ms (כניסות)
