סקילים חסרים שכדאי ליצור

סקילים אלה לא קיימים עדיין ויצירתם תשפר משמעותית את העקביות והאיכות בפרויקטים. כל סקיל מוגדר עם מטרה, תוכן מוצע ופרויקטים שיירוויחו.
character-consistency
מנגנון עקביות דמויות - reference-based generation עם model sheets, style guides, expression sheets. וידוא שכל דמות נראית אותו דבר בכל מקום.
תוכן מוצע: תבנית לדמות (model sheet + color palette + proportions + expressions), כללי prompt engineering עם ref images, בדיקת עקביות אוטומטית, תהליך review/correction.
חסר קריטי Game Art
game-ui-patterns
דפוסי UI למשחקים - HUD, Health Bars, Progress Indicators, Achievement Popups, Inventory, Dialog Boxes, Battle UI, Leaderboards, Mini-maps.
תוכן מוצע: קומפוננטות משחקיות מוכנות (XP bar, combo counter, damage numbers), אנימציות game-specific (screen shake, flash), accessibility במשחקים, responsive game UI.
חסר קריטי Game UI
hebrew-rtl-patterns
כללי RTL ועברית לכל פרויקט - CSS Logical Properties, bidirectional content, Hebrew font stacks, mixed content (Hebrew+English), number formatting.
תוכן מוצע: מיפוי ms/me/ps/pe מלא, כללי direction לאלמנטים ספציפיים (code blocks, numbers), Hebrew date formatting, Gemini prompt templates לעברית, בדיקות RTL אוטומטיות.
חסר קריטי i18n
sound-design-patterns
דפוסי עיצוב סאונד למשחקים - Web Audio API, SFX preloading, music management, volume layers, spatial audio, adaptive soundtrack.
תוכן מוצע: Audio sprite sheets, crossfade patterns, SFX pooling, context-aware music (battle/exploration/menu), ElevenLabs TTS integration, Suno music pipeline.
חסר בינוני Audio
story-engine-patterns
דפוסי מנוע סיפור אינטראקטיבי - branching narratives, dialog systems, character mood states, cutscenes, quest tracking, save points.
תוכן מוצע: Script format (beats + choices), DialogBox patterns, StoryPlayer FSM, character emotion system, speech bubble styling, Hades-style avatar dialog, cinematic sequences.
חסר בינוני Narrative
gemini-asset-pipeline
Pipeline ליצירת assets עם Gemini - batch generation, style consistency, naming conventions, directory structure, optimization, format conversion.
תוכן מוצע: תהליך batch ליצירת סט דמויות/רקעים/אייקונים, prompt templates לסגנונות שונים, post-processing (resize, optimize, webp), ארגון תיקיות, manifest file ל-assets.
חסר בינוני Asset Pipeline

המלצות ליישום רוחבי

המלצות אלה מתייחסות ל-כל 32 הפרויקטים של המשתמש - לא רק נינג'ה מקלדת. הסקילים מחולקים ל-3 רמות: Core (חובה לכולם), Recommended (מומלץ לרוב), Specialized (לפי צורך).

Core - חובה בכל פרויקט

ui-skills + baseline-ui (מיזוג)

שני הסקילים כמעט זהים אך baseline-ui מוסיף כללים חשובים (z-index scale, text-balance, tabular-nums). מומלץ למזג אותם לסקיל אחד שישמש כסטנדרט עבור כל הפרויקטים.

פרויקטים: כל 32 הפרויקטים עם UI

nano-banana-poster (Gemini Images)

כל פרויקט צריך תמונות. Gemini חינמי ומייצר תוצאות מעולות. כלל ברזל: אף placeholder לא יישאר - כל תמונה תיוצר עם Gemini.

פרויקטים: ninja-keyboard, omanut-hakesher-website, portfolio-website, bayit-beseder, ey.ai-kids-playground, eitan-events, date-kirva, haderech-next, elad-score, content-studio, financial-manager

google-stitch + shadcn-ui (Design Pipeline)

Stitch לעיצוב ראשוני, shadcn/ui כבסיס קומפוננטות. יחד הם יוצרים pipeline שלם מעיצוב לקוד. לוודא שמותקן ב-כל פרויקט React/Next.js.

פרויקטים: כל פרויקטי React/Next.js (20+)

web-accessibility + fixing-accessibility

שני סקילים משלימים: web-accessibility מספק דפוסי ARIA ו-Keyboard Nav, fixing-accessibility מספק ביקורת ממוקדת עם 9 קטגוריות עדיפות. יחד = כיסוי מלא של WCAG 2.2 AA.

פרויקטים: כל 32 הפרויקטים (חובה חוקית!)

hebrew-rtl-patterns (ליצירה!)

רוב הפרויקטים בעברית. אין סקיל מרכזי שמנהל את כללי RTL, CSS Logical Properties, Hebrew fonts ו-mixed content. זה הסקיל הכי קריטי ליצירה - יחסוך שגיאות חוזרות בכל פרויקט.

פרויקטים: ninja-keyboard, bayit-beseder, omanut-hakesher-*, haderech-next, eitan-events, date-kirva, financial-manager, content-studio, elad-score, hitechkids, edutech, mens-circle ועוד

Recommended - מומלץ לרוב הפרויקטים

vercel-react-best-practices + composition-patterns

45 כללי ביצועים מ-Vercel + דפוסי הרכבה. יחד מונעים waterfalls, bundle bloat ו-boolean prop proliferation. חובה בכל פרויקט React/Next.js.

פרויקטים: כל פרויקטי Next.js (ninja-keyboard, haderech-next, portfolio, omanut-hakesher-website ועוד)

fixing-metadata

Metadata נכון = SEO טוב = משתמשים. Title, OG tags, canonical, favicons, JSON-LD. חובה לכל אתר ציבורי - רוב הפרויקטים חסרים metadata תקין.

פרויקטים: omanut-hakesher-website, portfolio-website, eitan-events, haderech-next, content-studio, creative-workshops, drumming-workshops

web-design-guidelines (Vercel)

ביקורת דינמית שמושכת כללים עדכניים מ-GitHub. משלים את ui-skills עם פרספקטיבה של Vercel Labs. להפעיל כ-post-review על כל UI חדש.

פרויקטים: כל פרויקטי web (20+)

ui-ux-pro-max (Design Intelligence DB)

מאגר חיפוש עם 50+ סגנונות, 97 פלטות צבעים ו-57 זוגות פונטים. מצוין בתחילת כל פרויקט חדש לבחירת כיוון עיצובי מבוסס נתונים.

פרויקטים: פרויקטים חדשים, redesign, omanut-hakesher, date-kirva, eladjak-hub

design-motion-principles + fixing-motion-performance

ביקורת אנימציה (3 מעצבים) + תיקון ביצועים (composite vs paint vs layout). חובה לכל UI עם אנימציות, במיוחד משחקים ואפליקציות אינטראקטיביות.
