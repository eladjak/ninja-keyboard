שורות בדיקה (4 שורות)

כל שורה נוצרת ב-4 גרסאות: raw, tempo+10%, tempo+15%, energized

מזהה טקסט בעברית הקשר
yuki-challenge !בואו נראה מי הכי מהיר כאן אתגר מהירות - פתיחה
yuki-win !שברתי את השיא! אף אחד לא יעקוף אותי ניצחון - אחרי שבירת שיא
yuki-encourage !מהר יותר! מהר יותר! אני יכולה לנצח עידוד - במהלך תרגול
yuki-taunt ?אתה חושב שאתה מהיר? תראה את זה פרובוקציה - אתגר ישיר

צינור עיבוד ffmpeg

כל שורת בדיקה עוברת 3 סוגי עיבוד נוספים מעבר ל-raw:

1
ElevenLabs TTS (Raw)
יצירת הקול הגולמי עם eleven_v3, language_code='he', הגדרות TTS לפי וריאציה
{line-id}-raw.mp3
2
Tempo +10%
האצת דיבור ב-10% בלי לשנות גובה צליל. שומר על טבעיות.
ffmpeg -i raw.mp3 -af "atempo=1.10" tempo110.mp3
3
Tempo +15%
האצה אגרסיבית יותר. עלולה להישמע מעט מלאכותית אבל מהירה מאוד.
ffmpeg -i raw.mp3 -af "atempo=1.15" tempo115.mp3
4
Energized (Tempo + EQ Boost)
האצה של 12% + חיזוק תדרים גבוהים (3kHz) לתחושת בהירות ואנרגיה. מוסיף "נוכחות" לקול ומדגיש ניואנסים אקספרסיביים.
ffmpeg -i raw.mp3 -af "atempo=1.12,equalizer=f=3000:width_type=o:width=2:g=3" energized.mp3

איך להריץ

1
וידוא שמפתח API זמין
הסקריפט משתמש במפתח ElevenLabs מתוך ELEVENLABS_API_KEY env var, קובץ .env בתיקיית scripts, או fallback מוטמע.
2
הרצת הסקריפט
node scripts/yuki-v8-generate.mjs
(זמן משוער: 5-8 דקות, תלוי בזמני תגובה של ElevenLabs)
3
האזנה להשוואה
האזן לכל הקבצים ב-v8-previews/var-*/. חפש את השילוב הטוב ביותר של: מבטא ישראלי נקי, מהירות, ניואנסים רגשיים.
4
בחירה והעתקה
העתק את הקבצים הזוכים לתיקייה הראשית yuki/ ועדכן את account-voices-v4.json.

שלבים הבאים

  • הריצ/י את הסקריפט ליצירת כל הקולות
  • האזנה לכל ה-previews ובחירת הוריאציה הטובה ביותר
  • בחירת רמת tempo מועדפת (raw / +10% / +15% / energized)
  • העתקת הקבצים הזוכים לתיקייה הראשית
  • עדכון account-voices-v4.json עם ה-voice ID המנצח
  • אם אף וריאציה לא מספיק טובה - שינוי תיאורים וניסיון V9
  • יצירת שורות נוספות ליוקי (battle, unlock, story beats)
