🚨 בעיות נגישות שנמצאו ▶

CRITICAL   oklch() — ניגודיות גבולות

ערכי oklch() בגבולות עשויים להיכשל בבדיקות ניגודיות אוטומטיות. אין השפעה מהותית כיוון שמדובר בגבולות דקורטיביים בלבד — אין טקסט על הגבולות. יש לוודא שכל טקסט עומד ביחס ניגודיות 4.5:1.

HIGH   prefers-reduced-motion לא מוטמע

אנימציות Framer Motion לא בודקות את prefers-reduced-motion. משתמשים הרגישים לתנועה עלולים לחוות אי-נוחות. נדרש לעטוף אנימציות עם hook useReducedMotion() מ-Framer Motion.

const reducedMotion = useReducedMotion()
const variants = {
  hidden: { opacity: 0, y: reducedMotion ? 0 : 20 },
  visible: { opacity: 1, y: 0 }
}

HIGH   aria-live לשינויי מצב משחק חסר

שינויים דינמיים (טיימר, ניקוד, עדכוני קצב הקלדה) לא מוכרזים לקוראי מסך. נדרשות אזורי aria-live="polite" לעדכונים שוטפים ו-aria-live="assertive" לאירועי ניקוד חשובים.

MEDIUM   חסר alt text תיאורי לתמונות

חלק מתמונות הדמויות חסרות טקסט חלופי תיאורי. יש להוסיף alt text בעברית לכל תמונות הדמויות ותמונות ה-UI.
