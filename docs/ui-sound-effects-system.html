העיקרון המנחה: כל אינטראקציה ראויה לתגובה קולית

משחקי מובייל מובילים כמו Brawl Stars, Clash Royale ו-Genshin Impact מלמדים אותנו שסאונד הוא לא תוספת - הוא חלק בלתי נפרד מחוויית המשחק. כשלוחצים על כפתור ולא שומעים כלום, המשחק מרגיש "מת". כשכל קליק מלווה בסאונד עדין ומספק - המשחק מרגיש חי ומגיב.

עדינות ושביעות רצון

  • סאונדים עדינים שלא מעצבנים בחזרות
  • מספקים - כל קליק צריך להרגיש "טוב"
  • קצרים - רוב הסאונדים 30-150ms
  • לא חוסמים - סאונד חדש לא ממתין לקודם

אופי ישראלי-גיימרי

  • תחושת נינג'ה: ווש, סלאש, שוריקן
  • טעם ישראלי: קליפי קול של הדמויות בעברית
  • אנרגיית משחק מובייל: סיפוק מיידי
  • השראה: Brawl Stars + Duolingo + Fruit Ninja

התאמת גיל

  • שתיל (6-8): סאונדים שובבים, מנגינתיים, עליזים
  • נבט (8-10): סאונדים אנרגטיים, הרפתקניים
  • גזע (10-12): סאונדים מגניבים, ניניג'ה-אסקיים
  • ענף (12-14): סאונדים חדים, מקצועיים
  • צמרת (14-16+): סאונדים מינימליסטיים, אלגנטיים

בקרת ווליום מפורטת

  • Master Volume: שליטה כללית על הכל
  • SFX Volume: אפקטים קוליים (קליקים, הישגים)
  • Music Volume: מוזיקת רקע (כבר קיים ב-MusicStore)
  • Voice Volume: קליפי קול של דמויות
  • כל קטגוריה עם Mute toggle עצמאי
כלל הזהב:

אם שחקן משחק שעה ולא מבחין בסאונד ספציפי - אבל מרגיש שמשהו חסר כשמכבים אותו - הסאונד הזה מושלם. סאונד UI טוב הוא תת-מודע.

Audio Pool - מניעת עומס זיכרון

במקום ליצור HTMLAudioElement חדש בכל play (כמו שהמערכת הנוכחית עושה עם cloneNode), ניצור Pool של אלמנטים מוכנים:

// src/lib/audio/audio-pool.ts

const POOL_SIZE = 8  // Max concurrent sounds per key

class AudioPool {
  private pools = new Map<string, HTMLAudioElement[]>()
  private index = new Map<string, number>()

  /** Pre-create a pool for a sound key. */
  createPool(key: string, src: string, size = POOL_SIZE): void {
    const pool: HTMLAudioElement[] = []
    for (let i = 0; i < size; i++) {
      const audio = new Audio(src)
      audio.preload = 'auto'
      pool.push(audio)
    }
    this.pools.set(key, pool)
    this.index.set(key, 0)
  }

  /** Play next available element from pool (round-robin). */
  play(key: string, volume: number): boolean {
    const pool = this.pools.get(key)
    if (!pool) return false

    const idx = this.index.get(key) ?? 0
    const audio = pool[idx]

    audio.currentTime = 0
    audio.volume = volume
    audio.play().catch(() => {})

    this.index.set(key, (idx + 1) % pool.length)
    return true
  }

  /** Release pool for a key (free memory). */
  release(key: string): void {
    this.pools.delete(key)
    this.index.delete(key)
  }
}

אסטרטגיית טעינה

  • eager (בטעינה): keyClick, correct, error, buttonClick, navigate, tabSwitch - סאונדים שמשתמשים בהם מיד
  • lazy (ברקע): levelComplete, achievementFanfare, battleStart - סאונדים שמשתמשים בהם אחרי כמה דקות
  • on-demand (בהזמנה): voice clips, story sounds - נטענים רק כשנכנסים לדף הרלוונטי

ביצועים - דרישות

  • זמן השמעה: <16ms (פריים אחד ב-60fps)
  • AudioContext: singleton, resume on user gesture
  • GainNode: אחד לכל קטגוריה (sfx, ui, voice, music)
  • Cooldown: מניעת spam (למשל hover - 60ms min)
  • maxConcurrent: הגבלת sounds חופפים (keyClick=5, levelUp=1)
  • זיכרון: ~2-4MB לכל הסאונדים (MP3 128kbps, 1-4 שניות)
// src/lib/audio/volume-controller.ts
// שליטה בווליום עם GainNode נפרד לכל קטגוריה

class VolumeController {
  private ctx: AudioContext
  private masterGain: GainNode
  private categoryGains: Map<SoundCategory, GainNode>

  constructor(ctx: AudioContext) {
    this.ctx = ctx
    this.masterGain = ctx.createGain()
    this.masterGain.connect(ctx.destination)

    this.categoryGains = new Map()
    for (const cat of ['sfx', 'music', 'voice', 'ui'] as const) {
      const gain = ctx.createGain()
      gain.connect(this.masterGain)
      this.categoryGains.set(cat, gain)
    }
  }

  /** Get the GainNode for a category (to connect sounds through). */
  getOutput(category: SoundCategory): GainNode {
    return this.categoryGains.get(category) ?? this.masterGain
  }

  /** Set volume for a category (0-1). */
  setVolume(category: SoundCategory | 'master', volume: number): void {
    const clamped = Math.max(0, Math.min(1, volume))
    if (category === 'master') {
      this.masterGain.gain.setValueAtTime(clamped, this.ctx.currentTime)
    } else {
      this.categoryGains.get(category)?.gain.setValueAtTime(clamped, this.ctx.currentTime)
    }
  }

  /** Mute/unmute a category. */
  setMuted(category: SoundCategory, muted: boolean): void {
    const gain = this.categoryGains.get(category)
    if (gain) gain.gain.setValueAtTime(muted ? 0 : 1, this.ctx.currentTime)
  }
}

useSoundEffect() Hook - ליבת האינטגרציה

Hook שמאפשר לכל קומפוננטה לנגן סאונדים בקלות, עם בדיקת הגדרות אוטומטית:

// src/hooks/use-sound-effect.ts

import { useCallback, useRef } from 'react'
import { soundManager } from '@/lib/audio/sound-manager'
import { useSettingsStore } from '@/stores/settings-store'

type SoundKey =
  | 'buttonClick' | 'buttonSecondary' | 'buttonHover'
  | 'navigate' | 'tabSwitch' | 'menuOpen' | 'menuClose'
  | 'keyClick' | 'correct' | 'error' | 'comboHit'
  | 'levelComplete' | 'xpGain' | 'badgeUnlock'
  | 'formError' | 'toastSuccess' | 'toastError'
  // ... all other sound keys

interface UseSoundOptions {
  /** Disable this specific sound (e.g. for reduced audio mode) */
  disabled?: boolean
  /** Override volume (0-1) */
  volume?: number
}

export function useSoundEffect(key: SoundKey, options?: UseSoundOptions) {
  const soundEnabled = useSettingsStore((s) => s.soundEnabled)
  const lastPlayed = useRef<number>(0)

  const play = useCallback(() => {
    if (!soundEnabled || options?.disabled) return

    // Cooldown check
    const now = Date.now()
    const config = UI_SOUNDS[key]
    if (config && now - lastPlayed.current < config.cooldownMs) return
    lastPlayed.current = now

    soundManager.playSound(key, options?.volume)
  }, [key, soundEnabled, options?.disabled, options?.volume])

  return { play }
}

// Shorthand for common patterns:
export function useClickSound() {
  return useSoundEffect('buttonClick')
}

export function useNavigateSound() {
  return useSoundEffect('navigate')
}

SoundProvider - הפעלה אוטומטית למערכת

// src/components/providers/sound-provider.tsx

'use client'

import { useEffect } from 'react'
import { soundManager } from '@/lib/audio/sound-manager'
import { useSettingsStore } from '@/stores/settings-store'

export function SoundProvider({ children }: { children: React.ReactNode }) {
  const soundEnabled = useSettingsStore((s) => s.soundEnabled)
  const soundVolume = useSettingsStore((s) => s.soundVolume)

  useEffect(() => {
    // Preload critical sounds on mount
    soundManager.preload()

    // Resume AudioContext on first user interaction
    const resume = () => {
      soundManager.resumeContext()
      document.removeEventListener('click', resume)
      document.removeEventListener('keydown', resume)
    }
    document.addEventListener('click', resume, { once: true })
    document.addEventListener('keydown', resume, { once: true })

    return () => {
      document.removeEventListener('click', resume)
      document.removeEventListener('keydown', resume)
    }
  }, [])

  // Sync settings store with sound manager
  useEffect(() => { soundManager.setEnabled(soundEnabled) }, [soundEnabled])
  useEffect(() => { soundManager.setVolume(soundVolume) }, [soundVolume])

  return <>{children}</>
}
אינטגרציה קיימת:

המערכת הנוכחית כבר טוענת ב-app-providers.tsx עם soundManager.preload(). ה-SoundProvider פשוט מחליף את הקריאה הנוכחית וגם מטפל ב-AudioContext resume ובסנכרון הגדרות.

סליידרי ווליום

  • Master: שליטה על כל הסאונדים (0-100%)
  • מוזיקה: מוזיקת רקע (כבר קיים ב-MusicStore)
  • אפקטים: SFX + UI sounds
  • קול: קליפי קול דמויות
  • כל סליידר עם כפתור mute/unmute
  • שמירה ב-localStorage דרך settings-store

מצבים מיוחדים

  • מצב מופחת (Reduced Audio): רק סאונדים קריטיים (הקלדה, הישגים). בלי hover, ניווט, פופ
  • מצב ויזואלי בלבד: כל הסאונדים כבויים, ויזואליזציות מחליפות (פלאש, רטט מסך)
  • רטט (Haptic, מובייל): navigator.vibrate() כתוספת או תחליף לסאונד
  • prefers-reduced-motion: בודק CSS media query ומפעיל מצב מופחת אוטומטית

  • כל סאונד אינפורמטיבי (הישג, שגיאה) מלווה ב-aria-live region
  • הודעות הצלחה/שגיאה תמיד עם טקסט, לא רק סאונד
  • Toast notifications עם role="alert"
  • סאונדים דקורטיביים (hover, ניווט) לא חוסמים שום מידע
