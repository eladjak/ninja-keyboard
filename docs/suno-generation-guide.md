## TASK FOR CLAUDE BROWSER EXTENSION

You are helping generate music tracks for a kids typing game called "Ninja Keyboard".

**Your job:** Go to **suno.com/create**, switch to **Advanced** mode and **v5** model, and generate each track listed below one by one. For each track:

1. Set the **Song Title** from the table
2. Paste the **Styles** tags into the Styles field
3. Type the **Exclude styles** into the Exclude styles field
4. Set the **Weirdness** slider to the specified percentage
5. Set the **Style Influence** slider to the specified percentage
6. **Leave Lyrics BLANK** — all tracks are instrumental
7. Set **Lyrics Mode** to **Manual**
8. Click **Create** and wait for 2 variations to generate
9. **Download BOTH variations** as MP3 files
10. Move to the next track

**Important notes:**
- Some tracks are marked `[v4.5]` — for those, switch the model dropdown to v4.5 before generating, then switch back to v5 for the next track
- Short jingles (5-10s) will generate as longer tracks — that's OK, we'll trim later
- If a generation fails, skip it and move to the next one
- Work through ALL 34 remaining tracks below in order

**DO NOT generate tracks marked ✅ — they are already done.**
