1 שרתי MCP - שירותי AI חיצוניים

▼
שרתי MCP (Model Context Protocol) הם שירותים חיצוניים שמתחברים ל-Claude Code ומרחיבים את היכולות שלו. כל שרת מספק כלים ייעודיים שניתן לקרוא להם ישירות מתוך סביבת העבודה.
Context7 פעיל
תיעוד ספריות עדכני - מאפשר לקרוא דוקומנטציה מעודכנת של ספריות JavaScript/TypeScript, כולל Next.js, React, Tailwind ועוד. פותר את בעיית ה-knowledge cutoff של מודלי AI.
resolve-library-id query-docs
שימוש: לפני כתיבת כל קוד - בדיקת syntax ו-API עדכני. חיוני במיוחד ל-Next.js 15 App Router, React 19, Tailwind CSS 4.
Octocode פעיל
חיפוש קוד אמיתי ב-GitHub - מאפשר למצוא implementations אמיתיים מפרויקטים בפרודקשן. משלים את Context7 עם דוגמאות מעשיות מהעולם האמיתי.
githubSearchCode githubGetFileContent githubViewRepoStructure githubSearchRepositories githubSearchPullRequests packageSearch
שימוש: חיפוש דפוסי שימוש אמיתיים, השוואת implementations, מציאת best practices. משולב עם Context7 לקבלת תמונה מלאה.
DeepWiki פעיל
תיעוד ריפוזיטורי GitHub מונע AI - שואל שאלות על ריפוזיטוריז ספציפיים ומקבל תשובות מבוססות על הקוד והתיעוד שלהם.
read_wiki_structure read_wiki_contents ask_question
שימוש: הבנה מעמיקה של ריפוזיטוריז ספציפיים כמו shadcn/ui, Zustand, next-intl. גיבוי ל-Context7.
Stitch (Google) פעיל
עיצוב UI מונחה AI מ-Google Labs. מייצר עיצובים, מסכים וקוד HTML/Tailwind ישירות. חובה לכל עבודת UI לפני כתיבת קוד.
build_site get_screen_code get_screen_image
שימוש: עיצוב מסכים חדשים, בניית אתרים מרובי עמודים, יצירת קומפוננטות UI. משולב עם סקילי Stitch (react-components, design-md).
ElevenLabs פעיל סקריפט
יצירת קולות AI לדמויות המשחק. משתמשים ב-Voice Design v3 API ליצירת קולות ייחודיים לכל דמות, עם עיבוד ffmpeg לאפקטים מיוחדים.
Voice Design v3 API Text-to-Speech ffmpeg post-processing
שימוש: 10 דמויות עם קולות מותאמים, 40+ שורות דיאלוג בעברית, אפקטים כמו pitch/echo/tremolo. יוקי כרגע ב-V7.
Exa זמין
חיפוש אינטרנט מתקדם מונע AI - חיפוש סמנטי, מחקר חברות, crawling של אתרים, ומחקר עמוק עם סוכנים.
web_search_exa web_search_advanced_exa crawling_exa deep_researcher_start deep_researcher_check company_research_exa people_search_exa get_code_context_exa
שימוש: מחקר מתחרים, חיפוש פתרונות טכניים, ניתוח שוק משחקי הקלדה לילדים.
Skill Registry זמין
רישום וניהול סקילים - מאפשר חיפוש, המלצות ואימות של סקילים זמינים ב-Claude Code.
list_skills search_skills get_skill recommend_skill validate_skill refresh_registry
שימוש: גילוי סקילים חדשים, אימות תקינות סקילים קיימים, המלצות לסקילים רלוונטיים למשימה.
Team MCP (OMC) זמין
תיאום צוותי סוכנים - מאפשר הרצת צוותים של סוכני Claude Code במקביל, עם ניהול סטטוס ותיאום משימות.
omc_run_team_start omc_run_team_wait omc_run_team_status omc_run_team_cleanup
שימוש: משימות מרובות-סוכנים כמו refactoring מקביל, סקירות קוד מרובות, build + test + lint בו-זמנית.

2 סוכני OMC - Oh-My-ClaudeCode

▼
Oh-My-ClaudeCode (OMC) היא שכבת אורקסטרציה מרובת-סוכנים ל-Claude Code. כל סוכן מתמחה בתחום ספציפי ומשתמש במודל AI שמתאים למורכבות המשימה: Haiku למשימות קלות, Sonnet למימוש, Opus לארכיטקטורה ואנליזה עמוקה.
BUILD נתיב בנייה וניתוח
explore haiku
סריקת קוד פנימית, מיפוי קבצים וסמלים. משמש לגילוי ראשוני של מבנה הפרויקט.
analyst opus
ניתוח דרישות, בהירות קריטריונים, זיהוי אילוצים נסתרים. מפרק דרישות עסקיות לטכניות.
planner opus
תכנון רצף משימות, יצירת תוכניות ביצוע, זיהוי סיכונים. מוודא שהעבודה מסודרת לפני מימוש.
architect opus
עיצוב מערכת, גבולות רכיבים, ממשקים, החלטות ארוכות-טווח. אחראי על ארכיטקטורה כוללת.
debugger sonnet
ניתוח שורש-בעיה, בידוד רגרסיות, אבחון כשלים. מומחה באיתור ותיקון באגים.
executor sonnet
מימוש קוד, רפקטורינג, עבודת פיצ'רים. הסוכן העיקרי לכתיבת קוד בפרויקט.
deep-executor opus
משימות אוטונומיות מורכבות, ביצוע מכוון-מטרה. לפיצ'רים גדולים שדורשים חשיבה עמוקה.
verifier sonnet
אימות השלמה, בדיקת טענות, ראיות לנכונות. מוודא שהעבודה באמת הושלמה כנדרש.
REVIEW נתיב סקירה
quality-reviewer sonnet
פגמי לוגיקה, תחזוקתיות, anti-patterns, פורמט, שמות, ביצועים, מורכבות. סקירת איכות מקיפה.
security-reviewer sonnet
פגיעויות, גבולות אמון, אימות/הרשאות. חיוני לפרויקט עם משתמשים ילדים (RLS, הסכמת הורים).
code-reviewer opus
סקירה מקיפה חוצת-תחומים, חוזי API, גרסאות, תאימות לאחור. הסוכן הכי מקיף לסקירה.
SPECIALIST מומחי תחום
test-engineer sonnet
אסטרטגיית בדיקות, כיסוי קוד, חיזוק בדיקות לא יציבות. מנהל את 1,093 הטסטים.
build-fixer sonnet
תיקון שגיאות בנייה, toolchain, שגיאות TypeScript. מתמחה בהחזרת ה-build לעבודה.
designer sonnet
ארכיטקטורת UX/UI, עיצוב אינטראקציות. עוזר בעיצוב חווית המשתמש ל-5 קבוצות גיל.
writer haiku
תיעוד, הערות מיגרציה, הנחיות למשתמש. כותב את 24 מסמכי ה-HTML של הפרויקט.
document-specialist sonnet
חיפוש תיעוד חיצוני ומסמכי עזר. מומחה ב-APIs, SDKs ותיעוד ספריות.
qa-tester sonnet
אימות CLI/שירות אינטראקטיבי. בודק שהמשחק עובד נכון בזמן ריצה.
critic opus
אתגור ביקורתי של תוכניות ועיצובים. מוצא חורים בלוגיקה ובקבלת החלטות.
תרשים שימוש בפרויקט
תהליך סוכנים תדירות
פיתוח פיצ'ר analyst → planner → executor → test-engineer → quality-reviewer → verifier גבוהה
חקירת באג explore + debugger + executor + test-engineer + verifier גבוהה
סקירת קוד quality-reviewer + security-reviewer + code-reviewer בינונית
תכנון ארכיטקטורה architect + planner + critic בינונית
יצירת תוכן designer + writer + executor נמוכה

3 סקילים בשימוש בפרויקט

▼
סקילים הם פקודות מיוחדות שמפעילים זרימות עבודה מוגדרות מראש. כל סקיל מביא ידע ודפוסים ספציפיים לתחום שלו.
WORKFLOW תזמור וביצוע
autopilot
ביצוע אוטונומי מלא מרעיון לקוד עובד. מזהה, מתכנן, מממש, בודק ומאמת באופן אוטומטי.
team
N סוכני Claude מתואמים עם pipeline שלבי: plan → prd → exec → verify → fix. מנגנון ברירת המחדל למשימות מורכבות.
ultrawork (ulw)
מקסימום מקביליות עם תזמור סוכנים מקביל. לעבודה שדורשת מהירות מקסימלית.
ralph
לולאת התמדה עם אימות - ממשיך לעבוד עד שהמשימה מושלמת ומאומתת. כולל ultrawork.
pipeline
שרשור סוכנים עם העברת נתונים בין שלבים. לתהליכים ליניאריים מובנים.
plan
תכנון אסטרטגי עם תמיכה ב-consensus ו-review. מצב ralplan למחלוקות מורכבות.
ultrapilot
חזית תאימות מעל Team - ממפה ל-pipeline השלבי. לבנייה מקבילית.
ralplan
תכנון איטרטיבי עם Planner, Architect, Critic עד קונצנזוס. למשימות בסיכון גבוה.
QUALITY איכות קוד
code-review
סקירת קוד מקיפה עם code-reviewer. מזהה פגמי לוגיקה, anti-patterns, בעיות ביצועים.
better-result-adopt
מעבר ל-Result types מ-better-result. מחליף try/catch בלוגיקה עסקית ב-Result.ok/Result.err.
composition-patterns
דפוסי הרכבה ב-React - compound components, render props, hooks composition.
systematic-debugging
דיבאג ב-5 שלבים: זהה → שחזר → בודד → תקן → אמת. גישה שיטתית לאיתור באגים.
verification-before-completion
אימות מלא לפני סיום - טיפוסים, לינט, בדיקות, כיסוי. חובה לפני "done".
requesting-code-review
הגשת קוד לסקירה - מכין את הקוד ומפעיל סוכני סקירה רלוונטיים.
receiving-code-review
עיבוד פידבק מסקירת קוד - מטפל בהערות CRITICAL, HIGH, MEDIUM לפי סדר עדיפות.
UI/DESIGN עיצוב וממשק משתמש
ui-skills
אילוצי UI כלליים - נגישות, קומפוננטות, אנימציות. בסיס לכל עבודת UI.
nano-banana-poster
יצירת תמונות עם Gemini (חינם!) - heroes, איורים, אייקונים, תגים. 14 model sheets + 8 רקעים יוצרו עם הכלי הזה.
stitch-loop
יצירת אתרים מרובי-עמודים מפרומפט אחד דרך Google Stitch MCP.
design-motion-principles
עקרונות אנימציה - transform/opacity בלבד, מקסימום 200ms לפידבק.
react-components
המרת עיצובי Stitch לקומפוננטות React. יוצר קוד TSX מוכן לשימוש.
enhance-prompt
שיפור פרומפטים ל-Stitch - הופך רעיונות UI לפרומפטים אופטימליים.
shadcn-ui
אינטגרציית shadcn/ui - קומפוננטות בסיסיות עם New York variant. עיצוב RTL מותאם.
design-md
תיעוד עיצוב מפרויקטי Stitch. מתעד החלטות עיצוביות ודפוסים.
ui-ux-pro-max
סוכן מעצב-מפתח UI/UX מתקדם. לעיצוב חוויות משתמש מורכבות.
TESTING בדיקות ואיכות
test-driven-development
זרימת TDD: RED → GREEN → REFACTOR. כותב טסט ראשון, ממש מינימלי, משפר.
agent-browser
בדיקות דפדפן אוטומטיות - ניווט, צילומי מסך, אינטראקציה עם אלמנטים. Playwright מתחת למכסה.
vitest-patterns
דפוסי Vitest - מבנה טסטים, mocking, snapshot testing. 1,093 טסטים ב-62 suites.
FRAMEWORKS פריימוורקים וספריות
nextjs-patterns
Next.js 15 App Router - Server Components, Route Groups, Middleware, Metadata API.
react-best-practices
React 19 + Next.js - Hooks, Suspense, Server Components, Concurrent Features.
zustand-patterns
ניהול state עם Zustand - slices, selectors, middleware. 6 stores בפרויקט.
tailwind-patterns
Tailwind CSS 4 - CSS logical properties, custom themes, RTL utilities, dark mode.
framer-motion-patterns
אנימציות Framer Motion - variants, AnimatePresence, layout animations. transform/opacity בלבד.
GIT/DEVOPS Git ותפעול
commit
יצירת commits עם conventional commits: feat/fix/refactor/docs/test/chore.
using-git-worktrees
פיתוח מבודד ב-git worktrees - כל פיצ'ר בסביבת עבודה נפרדת.
finishing-a-development-branch
סיום branch פיתוח - cleanup, squash, PR, merge. זרימה מסודרת.
writing-plans
כתיבת תוכניות עבודה מובנות - שלבים, תלויות, סיכונים, קריטריוני הצלחה.
executing-plans
ביצוע תוכניות עבודה - מעקב אחר שלבים, סימון התקדמות, דיווח.
dispatching-parallel-agents
שיגור סוכנים מקביליים - הפצת תת-משימות עצמאיות ל-2+ סוכנים בו-זמנית.
CONTENT יצירת תוכן
nano-banana-poster
יצירת תמונות עם Gemini Image API (חינם!) - heroes, איורים, model sheets, רקעים, UI assets.
fal-ai
יצירת וידאו ותמונות מתקדמות - image-to-image, ControlNet, background removal. לכשצריך מעבר ל-Gemini.
remotion
יצירת סרטוני walkthrough מעיצובים - הדגמת UI, תיעוד וידאו.
brainstorming
סיעור מוחות מובנה - רעיונות, הרחבות, אלטרנטיבות. לתכנון narrative ודמויות.

4 כלים ייחודיים לפרויקט

▼
כלים וסקריפטים שנוצרו במיוחד עבור פרויקט נינג'ה מקלדת - מנועי משחק, מערכות סאונד, מחוללי קולות ועוד.
1
voice-generate-all-hebrew.mjs
סקריפט מרכזי ליצירת קולות עברית לכל דמויות המשחק
  • 10 דמויות עם קולות ייחודיים (ElevenLabs Voice Design v3)
  • 40+ שורות דיאלוג בעברית לכל דמות
  • עיבוד ffmpeg: pitch shifting, echo, tremolo, vibrato
  • Rex (pitch 95%), Bug (pitch 115%+vibrato), BugKing (pitch 85%+echo)
  • תיקיית פלט: public/audio/voices/
2
yuki-v7-generate.mjs
סקריפט ייעודי לקול יוקי - הצברית האולטימטיבית
  • 4 וריאציות (A-D) עם הגדרות קול שונות
  • 4 עיבודי ffmpeg לכל וריאציה
  • תיקון content filter של ElevenLabs
  • מיקוד: מבטא ישראלי צברי, מהיר, ללא גוון רוסי
3
nano-banana-poster (Gemini Art Pipeline)
צינור יצירת אסטים ויזואליים עם Gemini Image API
  • 17 model sheets של דמויות (סגנון אחיד, מאושרים)
  • Expression sheets לכל דמות
  • 8 רקעי סביבה (דוג'ו, יער, עיר, חלל...)
  • UI assets: כפתורים, תגים, מסגרות
  • תיקיית פלט: public/images/
4
DialogBox + StoryPlayer
מנוע סיפור ודיאלוגים אינטראקטיבי
  • אפקט typewriter עם מהירות מותאמת
  • 4 סגנונות דיאלוג: רגיל, קריטי, חשיבה, הומור
  • בחירות שחקן עם השפעה על העלילה
  • 26 story beats בפרקים 1-3
  • מנגנון speech-bubble מונפש
5
useSoundEffect + Web Audio
מערכת סאונד היברידית - קבצי MP3 + סינתזה
  • 16 צלילי SFX מבוססי AI (קבצי MP3 ראשיים)
  • 12 צלילי UI מסונתזים (Web Audio API כגיבוי)
  • Preloading על mount - אפס דיליי
  • Zero dependencies - עובד ללא קבצים חיצוניים
6
AI Typing Engine
מנוע הקלדת AI לסימולציה של יריבים
  • סימולציה אנושית עם Gaussian jitter
  • Rubber-banding דינמי (מתאים קושי לשחקן)
  • 3 יריבים: Shadow (חתול), Storm (שועל), Blaze (דרקון)
  • פרופילי קושי מותאמים לגיל
7
MusicManager + MusicProvider
מערכת מוזיקה עם AudioContext וניהול מצבים
  • AudioContext + GainNode לשליטה בווליום
  • Crossfade אוטומטי בניווט בין עמודים
  • Suno Main Theme V2 = התימה הרשמית
  • Boss V1 = בוס ראשי, V2 = מיני בוס
  • מנגנון unlock מוזיקה כאצ'יבמנטים
