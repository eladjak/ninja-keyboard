פילוסופיית עיצוב הקולות

  1. מיפוי מהירות-דמות: מהירות הדיבור של כל דמות משקפת את האישיות שלה -- דמויות מהירות מדברות מהר יותר, דמויות חכמות מדברות לאט יותר.
  2. יציבות כתכונת אופי: הפרמטר stability ב-ElevenLabs משמש באופן יצירתי -- לגליץ' יציבות נמוכה מאוד (0.2) ליצירת דיבור בלתי צפוי ומגומגם, בעוד סנסיי זן בעל יציבות גבוהה (0.7) לרוגע עקבי.
  3. עברית-מקורית: כל השורות בעברית. מודל eleven_v3 מטפל היטב בעברית עם קולות אנגליים מוכנים מראש.
  4. מותאם-גיל: הקולות נבחרו בהתאם לגילאי הדמויות -- דמויות צעירות משתמשות בקולות צעירים, צב-המאסטר הזקן משתמש בקול קשיש.

סקריפט ייצור הקולות נמצא ב-scripts/generate-voices.cjs. ליצירה מחדש או הוספת שורות חדשות:

# ערוך את מערך VOICE_LINES בקובץ scripts/generate-voices.cjs # ואז הרץ: node scripts/generate-voices.cjs

הסקריפט באופן אוטומטי:

  • יוצר תיקיות לפי הצורך
  • מייצר שורה אחת בכל פעם עם השהיות של 2 שניות (בטיחות הגבלת קצב)
  • מדווח הצלחה/כישלון לכל שורה
  • מציג את כל הקבצים שנוצרו עם גדלים בסיום
