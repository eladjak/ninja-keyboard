# Ninja Keyboard - Character World Bible

## Vision
A colorful, fun gaming world where kids and adults enjoy learning to type.
Every character has personality, voice, story, and a role in the typing journey.
Style: **Brawl Stars + Anime + Disney/Pixar** - chibi proportions, expressive eyes, bold colors.

## Environments

### Training Dojo (דוג'ו אימונים)
- **Used in**: Lessons, basic practice
- **Theme**: Traditional Japanese dojo, wooden floors, paper screens, cherry blossoms
- **Background**: Warm amber lighting, scrolls on walls with Hebrew letters
- **Characters present**: Ki, Sensei Zen

### Speed Arena (זירת המהירות)
- **Used in**: Speed tests, timed challenges
- **Theme**: Futuristic racing track, neon lights, countdown timers
- **Background**: Dark with blue/cyan neon streaks, digital displays
- **Characters present**: Yuki, Rex

### Battle Colosseum (קולוסיאום הקרב)
- **Used in**: Battle mode vs AI/players
- **Theme**: Epic arena, crowd cheering, dramatic lighting
- **Background**: Dark arena with purple/red spotlights, spectator silhouettes
- **Characters present**: Kai, Shadow/Storm/Blaze

### Cyber Lab (מעבדת הסייבר)
- **Used in**: Keyboard shortcuts, tech challenges
- **Theme**: High-tech hacker lab, holographic displays, green code streams
- **Background**: Dark with matrix-style code rain, floating screens
- **Characters present**: Mika, Pixel

### Moonlit Garden (גן הירח)
- **Used in**: Free practice, meditation typing
- **Theme**: Peaceful Japanese garden at night, fireflies, moonlight
- **Background**: Deep blue with soft moonlight, cherry blossom petals falling
- **Characters present**: Luna, Noa

### Games Arcade (ארקייד המשחקים)
- **Used in**: Games hub (Word Rain, Ninja Slice, Letter Memory)
- **Theme**: Retro arcade with modern touches, glowing cabinets
- **Background**: Colorful neon signs, pixel art decorations
- **Characters present**: Rex, Pixel
