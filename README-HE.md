# נינג'ה מקלדת (Ninja Keyboard)

מאמן הקלדה עברי עם גיימיפיקציה שהופך את לימוד המקלדת לכיף. תרגלו הקלדה עם רמת קושי מתאימה, עקבו אחרי ההתקדמות שלכם והתחרו באחרים -- הכל בחוויית נינג'ה עם דמויות מקוריות, קריינות ואפקטים קוליים.

## תכונות

- רמת קושי מותאמת אישית
- מעקב מהירות הקלדה (WPM) בזמן אמת
- מצב מרובה משתתפים
- דמויות נינג'ה מקוריות עם קריינות
- אפקטים קוליים ואנימציות
- לוח תוצאות ודירוג
- תמיכה מלאה בעברית

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| [Next.js 16](https://nextjs.org/) | פריימוורק React |
| [TypeScript](https://www.typescriptlang.org/) | בטיחות טיפוסים |
| [Supabase](https://supabase.com/) | בקאנד, אימות ומסד נתונים |
| [Tailwind CSS](https://tailwindcss.com/) | עיצוב |
| [Framer Motion](https://www.framer.com/motion/) | אנימציות |

## התקנה

```bash
git clone https://github.com/eladjak/ninja-keyboard.git
cd ninja-keyboard
bun install
bun dev
```

פתחו [http://localhost:3000](http://localhost:3000) בדפדפן.
